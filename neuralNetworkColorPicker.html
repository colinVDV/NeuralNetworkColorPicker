Neural Network Color Picker

Welcome to a demo of my first project incorporating a neural network. For this project I implemented "Brain.Js". Since this was my first Neural network project I decided to pick something simple. I trained this neural net so that it would recognize which text-color would work best with a certain color background. The neural network has been shown 16 different examples and from these fourteen examples, it calculates a result for every possible color. The calculation it makes is based off of the RGB color code. you can see the output of its calculation below our box. When the result is above 0.5, the text color should be white, otherwise, it should be black. Scroll down to learn a little more about the neural network that is being used here.

Neural Network Visualization

Above you can see a visualization of the neural network that is used for this project. On the left (green), we have three input layers representing red green and blue. On the right(blue), we have our output layer, this is the result of the neural network's calculation. In the middle (red), we have our "hidden layers", or as I like to call them, the computational layers. This is where the neural net does its calculations. I can manually set the number of nodes in our hidden layers, however, it is a good rule of thumb to use a number inbetween our input nodes and output nodes.
